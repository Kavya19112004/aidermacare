# 📱 Aidermacare Mobile Application

Aidermacare is a mobile application developed using React Native (Expo) to provide smart and user-friendly features for healthcare/assistance-related services.

---

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/Kavya19112004/aidermacare.git
```

### 2. Navigate to the mobile folder

```bash
cd mobile
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the application

```bash
npx expo start
```

---

## 📱 Run the App

* Scan the QR code using Expo Go (Android)
* OR press `a` to run on Android emulator

---

## 📂 Project Structure

* `app/` → Application source code
* `assets/` → Images, audio, and other resources
* `package.json` → Project dependencies

---

## 🛠️ Technologies Used

* React Native (Expo)
* Expo Router
* JavaScript
